Sistema de Gestión Hospitalaria

Este es un sistema de prueba desarrollado para gestionar citas médicas y administración hospitalaria.

Tecnologías Utilizadas:

  • Backend: FastAPI (Python)
  • Frontend: Django (Python) - Solo para pruebas
  • Base de Datos: MySQL
  • Autenticación: JWT Tokens

Funcionalidades:

  • Registro y autenticación de usuarios
  • Sistema de roles (Paciente, Médico, Administrador)
  • Gestión de citas médicas
  • Administración de médicos
  • Control de disponibilidad médica
  • Historial de cambios en citas

Endpoint FastAPI:

Backend corriendo en: http://127.0.0.1:8000

Documentación API: http://127.0.0.1:8000/docs
